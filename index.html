Canteen Management System

This C project is a Canteen management system that streamlines food ordering and administrative tasks. It features two main user roles: Manager and Customer. Customers can browse the menu, place orders, and generate bills. Managers, after password authentication, can edit the menu, update prices, reset settings, and check past orders by name, bill number, or date history. The system includes security measures like password changes and structured navigation for efficient operations. It enhances restaurant workflow by digitalizing order processing and management. The project is ideal for small restaurants seeking an organized, user-friendly system to handle customer orders and menu modifications efficiently.
