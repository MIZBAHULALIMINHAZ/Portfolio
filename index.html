IIUC Shuttle Management System

Tech Stack: Django, HTML ,Leaflet.js
The IIUC Shuttle Management System is a software application designed to streamline and optimize the shuttle service for students, staff, and visitors at the International Islamic University Chittagong (IIUC). It allows users to view shuttle schedules, track real-time shuttle locations, and book rides in advance. The system helps manage shuttle routes, driver assignments, and passenger capacity, ensuring efficient and timely transportation within the campus. Additionally, the system provides notifications for schedule updates, delays, and cancellations, enhancing the overall user experience. With this platform, IIUC aims to improve campus mobility and reduce transportation-related hassles for the university community.
